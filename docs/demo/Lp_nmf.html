L_p-Norm Non-Negative Matrix Factorization and Its Application to Singing Voice Enhancement

Tomohiko Nakamura (The University of Tokyo) and Hirokazu Kameoka (The University of Tokyo/NTT Communication Science laboratories)

We show a singing voice enhancement system using a non-negative matrix factorization with a L_p-norm criterion [1] All input sounds with an input SNR of 0 dB are taken from MIR-1K dataset [2].
